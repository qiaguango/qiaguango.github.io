# 项目目标
设计和开发此产品的目的是什么，请分别介绍一个为用户创造价值的目标，和一个商业化的目标。
比如:
1. 为了让社区的成员可以更好的分享与参与社区的活动；
2. App可以更好的聚合用户，提升用户活动度；

# 项目范围
- 是针对iPhone还是Android手机？
比如：只涉及iPhone

- 是针对国内市场还是国外市场？
比如：只有国内市场

- 产品主要满足用户那些TOP需求或问题？比如：
1. 可以通过微博或者微信注册；
2. 免注册浏览？
- 目标用户群的大致特征：
1. 90后大学生，渴望成长
2. 职场女性


# 竞争与商业
- 期望可以发展到多少用户量级？
比如1000万？

- 做的最好的三个竞争产品是哪些？

- 竞争产品体验的优点和缺点有哪些？


# 执行
- 此项工作的时间周期是怎样的？是否有明确的截止时间？
比如希望在2～3个月完成开发

- 此次工作有哪些人参与？分别承担怎样的工作？谁是最高决策者？
比如需要产品经理，运营负责人等分别是谁？
